##提示

分包：

    base        :组件基础包 如 BaseActivity，BaseFragment
    config      :参数配置包
    constants   : 常量
    db          :持久化
    http        :网络框架
    main        :主包，按功能模块分子包
    net         :网络请求
    update      :更新框架
    user        :账户控制
    utils       :基础工具包
    view        :自定义 View
    widget      :自定义组件
